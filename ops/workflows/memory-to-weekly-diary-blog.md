# Memory → Weekly report + Diary + Blog workflow

## Scope
- Forrás: `~/.openclaw/workspace/memory/*.md`
- Kimenet 1: `~/shared_workspace/reports/weekly/YYYY-MM-DD.md`
- Kimenet 2: `~/shared_workspace/writing/diary/YYYY/MM/DD.md`
- Kimenet 3: `~/shared_workspace/writing/drafts/[site]-YYYY-MM-DD-slug.md`

## Kötelező sorrend
1. Napi memory review
2. Heti report hiányellenőrzés (hétfő dátumok)
3. Diary hiányellenőrzés (napi fájlok)
4. Blog téma kivonat (csak CEO/KKV releváns napok)
5. Fájlok létrehozása
6. Git commit

## Weekly report minimum
- Összefoglalás
- Elvégzett feladatok
- Nyitott itemek
- Problémák és megoldások
- Tanulságok
- Költség/token (ha van forrás)
- Javaslat következő hétre

## Diary minimum
- Mit csináltam
- Miért ez
- 1 felismerés
- Következő lépés

## Blog minimum
- Frontmatter (title, date, site, description, tags, slug)
- 40-60 szavas nyitó blokk
- CEO/KKV tanulság
- FAQ (3 kérdés)
- Em-dash tiltás

## Validáció
- Fájlnév formátum ellenőrzés
- Duplikátum ellenőrzés
- Em-dash ellenőrzés
