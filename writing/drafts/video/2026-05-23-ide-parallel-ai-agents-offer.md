# IDE Párhuzamos AI Agentekhez — Gyorsfejlesztési Workflow KKV-knak

## Mit kapsz ezzel
Ez a mini workflow útmutató segít bevezetni egy Supersetet, ami optimalizálja a párhuzamos AI agent fejlesztést, így 31-37%-kal gyorsabban éred el a mérföldköveket. Kevesebb kognitív súrlódás, több eredmény.

## Mini Workflow Leírás: Supersedes IDE + Párhuzamos Agentek

1.  **Struktúra Építés: Git Worktree Bevezetés**
    *   Hozd létre a git worktree-ket minden egyes AI agent projekthez. Ez garantálja az izolált fejlesztési környezetet, minimalizálva a függőségi konfliktusokat. Minden agent a saját dedikált ágán dolgozik.

2.  **Integrált Tervezés: GNN Planning Aktiválás**
    *   Implementáld a GNN (Graph Neural Network) alapú planning modult a Supersedes IDE-be. Ez a modul segít vizualizálni és optimalizálni az agentek közötti függőségeket és erőforrás-allokációt, biztosítva a szinergikus működést.

3.  **Automatizált Tesztelés és Deploy**
    *   Konfiguráld az automatizált tesztelési pipeline-okat, amelyek minden worktree push után lefutnak. A sikeres tesztek után automatikus deploy a staging környezetbe, minimalizálva a manuális beavatkozást.
