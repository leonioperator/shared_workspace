# Reggeli AI Agent Csomag — KKV-k számára

## Mit kapsz ezzel

Beállítunk egy AI agentet, ami minden reggel automatikusan elvégzi a napi admin feladataidat:
ajánlat-követés, ügyfél-emlékeztetők, számlázási riport, napi összesítő.
Te reggel 5 percet szánsz rá — az agent elvégzi a többit.

---

## Workflow: Hogyan dolgozik a reggeli agent

### 1. Ajanlat-kovetes (automatikus)
- [ ] Agent figyeli az elküldött ajánlatokat
- [ ] 48 óra válaszhiány után automatikus emlékezteto emailt küld
- [ ] Te reggeli összesítot kapsz: ki reagált, ki nem

### 2. Ugyfel-emlekezteto rendszer
- [ ] Minden nyitott ügyfél folyamat nyomon követve
- [ ] Automatikus follow-up sablon beállítva (3/7/14 napos ciklus)
- [ ] Kézi beavatkozás csak akkor, ha az ügyfél válaszol

### 3. Napi összesíto (07:00-ra kész)
- [ ] Ajánlatok státusza
- [ ] Nyitott teendok prioritás szerint
- [ ] Számlázandó tételek listája
- [ ] Közelgo határidok (24-48 óra)

### 4. Számlázási riport
- [ ] Teljesített munkák automatikusan jelölve
- [ ] Havi összesíto generálása automatikusan
- [ ] Emlékezteto a késedelmes fizetoknek

### 5. Napi rutin kialakítása
- [ ] Agent setup: 2 munkanap
- [ ] Tesztelés: 3 próbanap veled együtt
- [ ] Átadás: te csak a reggeli 5 perces review-t csinálod

---

## Tipikus eredmény az elso heten

| Feladat | Korábbi idő | Agent után |
|---|---|---|
| Ajánlat-követés | 30-45 perc | 5 perc (review) |
| Ügyfél-emlékeztetok | 20-30 perc | 0 perc (auto) |
| Napi riport összeállítás | 20-30 perc | 0 perc (auto) |
| Számlázási ellenorzés | 15-20 perc | 5 perc (review) |
| **Összesen** | **85-125 perc** | **10 perc** |
